# Introduction to RL

**Key Words:** 
<!-- more -->

强化学习的特点：
使强化学习不同于其他机器学习范式的理由是：
    没有监督者，只有奖励信号
    反馈是延迟的，而不是即时的
    时间真的很重要（序列的、非 i.i.d 数据）
    Agent 的操作会影响它接收的后续数据

强化学习基于奖励假设
Definition (Reward Hypothesis)
All goals can be described by the maximisation of expected
cumulative reward
